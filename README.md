# EthaNET
Custom wireless protocol implemented in GNU radio, covering the PHY and MAC layers.

| MCS               | LEN      | SEQ    | DES    | SRC    | CRC    | PAYLOAD      |
| :---------------: | :------: | :----: | :----: | :----: | :----: | :----------: |
| 1 byte            | 1 byte   | 1 byte | 1 byte | 1 byte | 1 byte | 0-256 bytes  |
